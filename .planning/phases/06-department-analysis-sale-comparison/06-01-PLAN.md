  <behavior>
    Module exports:
    - `type DeltaDirection = 'up' | 'down' | 'flat' | 'none';`
    - `interface PairDelta { text: string; direction: DeltaDirection; }`
    - `function computePairDelta(current, previous, mode?)`
    - `function deltaColorClass(direction): string`

    `computePairDelta` contract:
    - Mode default `'relative'`.
    - If `current == null || previous == null` → `{ text: '—', direction: 'none' }` (em-dash U+2014).
    - Relative mode:
      - If `previous === 0` → `{ text: '—', direction: 'none' }` (divide-by-zero guard).
      - Compute `ratio = (current - previous) / previous`.
      - If `Math.abs(ratio) < 0.0005` (FLAT_THRESHOLD = 0.05% — locked by UI-SPEC) → `{ text: '0.0%', direction: 'flat' }`.
      - Otherwise: format `"+12.4%"` (positive) or `"-8.1%"` (negative, already has `-` from the number) using `(ratio * 100).toFixed(1)`. Direction `'up'` if ratio > 0 else `'down'`.
    - `absolute_pp` mode (both inputs are ratios in [0,1]):
      - Compute `diff = current - previous`.
      - If `Math.abs(diff) < 0.0005` → `{ text: '0.0pp', direction: 'flat' }`.
